# Arena

?> Represents a Clash Royale arena.

## Attributes

> **`name`** - The arena's name.    
**Returns:** `str`

> **`number`** - Which arena this arena is.    
**Returns:** `int`

> **`limit`** - The limit for the trophies.    
**Returns:** `int`

> **`imageUrl`** - URL for the image.    
**Returns:** `str`
